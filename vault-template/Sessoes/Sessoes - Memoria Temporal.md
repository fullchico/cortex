# Sessoes — Memoria Temporal

#sessoes #moc #memoria

> [!abstract] O cerebro cronologico do projeto
> Cada sessao de trabalho captura **decisoes tomadas**, **contexto discutido** e **artefatos gerados**. Isso permite que qualquer sessao futura reconstrua o estado mental do projeto sem repetir perguntas.

Voltar: [[Memoria Projeto]]

---

## Como usar

### Para o humano
Depois de cada sessao produtiva, execute `/fechar-sessao <VAULT>` para registrar automaticamente.

### Para o Claude (em sessoes futuras)
Antes de responder perguntas sobre o projeto, leia:
1. [[Memoria Projeto]] — visao geral e estado
2. A **ultima sessao** desta pasta — contexto mais recente
3. [[Definicoes Travadas]] — decisoes que nao devem ser rediscutidas
4. [[Regras Gerais]] — logica e formulas de negocio

---

## Convencao de nomes

`Sessao YYYY-MM-DD - Titulo descritivo`

Uma nota por dia. Se o trabalho continua no mesmo dia, fazer append na nota existente.
